Frequently asked questions

What is tervinOS based on?

tervinOS is built on a minimal Linux base with an LTS kernel carrying zen patches, tuned for responsiveness on desktops and gaming rigs alike.

Can I install other software?

Absolutely. tervinOS ships lean, but a full package manager and Flatpak support are included — install anything you like. We just won't pre-install it for you.

Will my Steam games run?

Anything that runs on Linux via Proton runs on tervinOS. Proton and Proton GE are pre-configured, and shader pre-caching is set up so games launch smoothly the first time.

What are the system requirements?

Desktop edition: 4 GB RAM, dual-core 64-bit CPU, 20 GB storage. Minimal edition runs comfortably on 2 GB RAM. Gaming edition benefits from a dedicated GPU.

Is it free?

Yes — free to download, use, and share, forever. tervinOS is open source and community-driven.
